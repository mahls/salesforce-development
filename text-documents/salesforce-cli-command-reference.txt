# Salesforce CLI Commands Reference

## Org Commands
- `sf org create` — Create a scratch org  
- `sf org list` — List all orgs  
- `sf org open` — Open a Salesforce org in the browser  
- `sf org delete` — Delete a scratch org  
- `sf org display` — Display details about a specific org  
- `sf org clone` — Clone an existing org  
- `sf org shape create` — Create org shape for shape-based scratch orgs  

## Data Commands
- `sf data soql query` — Run a SOQL query  
- `sf data record create` — Create a record  
- `sf data record update` — Update a record  
- `sf data record delete` — Delete a record  
- `sf data tree import` — Import data from JSON files  
- `sf data tree export` — Export data as JSON files  

## Apex Commands
- `sf apex execute` — Execute anonymous Apex code  
- `sf apex test run` — Run Apex tests  
- `sf apex log get` — Retrieve Apex debug logs  
- `sf apex log list` — List Apex debug logs  
- `sf apex log tail` — Stream Apex debug logs  

## Metadata Commands
- `sf source push` — Push source to scratch org  
- `sf source pull` — Pull source from scratch org  
- `sf source deploy` — Deploy source to org  
- `sf source retrieve` — Retrieve source from org  
- `sf source status` — Show changes between local and org  
- `sf source convert` — Convert source to metadata API format  
- `sf mdapi deploy` — Deploy metadata API package  
- `sf mdapi retrieve` — Retrieve metadata API package  
- `sf mdapi convert` — Convert metadata API format to source format  

## User Commands
- `sf user create` — Create a user in scratch org  
- `sf user display` — Display user info  
- `sf user password generate` — Generate user password  
- `sf user permset assign` — Assign permission set  

## Package Commands
- `sf package create` — Create a package  
- `sf package version create` — Create a package version  
- `sf package install` — Install package in org  
- `sf package uninstall` — Uninstall package from org  
- `sf package version list` — List package versions  

## Plugin Commands
- `sf plugins` — List installed plugins  
- `sf plugins install` — Install plugin  
- `sf plugins uninstall` — Uninstall plugin  
- `sf plugins update` — Update plugins  

## Auth Commands
- `sf auth web login` — Authenticate with org using web login  
- `sf auth logout` — Log out from org  

## Miscellaneous Commands
- `sf alias set` — Set an alias for an org  
- `sf alias list` — List aliases  
- `sf config set` — Set CLI config  
- `sf config get` — Get CLI config  
- `sf config list` — List CLI configs  
- `sf org status` — Show org status  
- `sf source tracking clear` — Clear source tracking
